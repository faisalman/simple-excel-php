simple-excel-php by faisalman

SimpleExcel is a PHP library with simplistic approach for parsing/writing data from/to Microsoft Excel XML format

Features === - Parsing Microsoft Excel 2003 XML Spreadsheet cells to array data - Writing array data to Microsoft Excel 2003 XML Spreadsheet cells

Dependencies

PHP5

Install

Just include this library on your PHP project
